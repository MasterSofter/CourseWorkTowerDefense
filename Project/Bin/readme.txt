Перед запуском программы убедитесь, что тексты программ закодированы кодировкой WIN1251.
Исходные файлы должны быть закодированы также в кодировке WIN1251.


Программа вызывается из консоли. Пример: Kursovaya.exe file.ANSI fileOut.ANSI 4
** Первый аргумент - входной файл
** Воторй аргумент - выходной файл
** Третий аргумент - длина искомых слов
----------
Программа написана на macOS в среде разработки JetBrains CLion 2018.3.4
